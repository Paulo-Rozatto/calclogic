O que é?

A Calc Logic é uma ferramenta para auxiliar estudantes de lógica matemática com o cálculo proposicional. Nela é possível operar com proposições como se estivesse usando uma calculadora tradicional para matemática.

Atualmente é possível se obter a tabela verdade de uma proposição inserida, mas futramente também será possível testar a validade de uma equivalência ou de um argumento.

O código da Calc Logic é aberto, sendo possível encontrá-lo no github.
